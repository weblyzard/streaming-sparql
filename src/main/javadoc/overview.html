Executes queries against a SPARQL server and returns a {@link ch.htwchur.isparql.StreamingResultSet} object that allows incremental processing and iterating over the query results.
The {@link ch.htwchur.isparql.StreamingQueryExecutor} performs a SPARQL query on a SPARQL repository and returns a {@link ch.htwchur.isparql.StreamingResultSet} iterator which allows incremental processing of streaming results as they are received by the server.

Example code:

  s = StreamingQueryExecutor.getResultSet("http://dbpedia.org/sparql", "SELECT ?s ?p ?o WHERE { ?s ?p ?o. } LIMIT 5")
  while (s.hasNext()) {
          System.out.println("Tupel " + s.getRowNumber() + ": " + s.next())
  }
